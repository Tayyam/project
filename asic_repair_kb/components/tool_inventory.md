## أولويات النواقص (من منظور ملفات الإصلاح)

1. **سلامة:** سوار تفريغ ساكن + تفضيل إضافة **سطح ESD** حتى مع وجود حصيرة الحرارة.
2. **S19 حسب الدليل:** **كرات 0.4 mm** + **سلك شفط (wick)** + كابل **IIC/EN** لبنش APW12.
3. **APW12:** معجون واجهة **2500** + سيليكون **704** + (اختياري) **حمل إلكتروني** لاختبارات الجودة تحت حمل.
4. **تسليم وتخزين:** أكياس ESD للعميل؛ كماشيات يدوية إن لم تكن موجودة في المعمل.

الصفوف ذات **✅** تبقى المرجع لما هو مُدخل أصلاً في `data.js`؛ راجع الخريطة هناك قبل الشراء.
